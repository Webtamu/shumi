# SHUMI

A growth & productivity app for tracking your practice sessions, visualizing progress, and building consistency.

---

## Setup instructions (Dev)

First, install uv for python package and version management

    # For mac/linux
    curl -LsSf https://astral.sh/uv/install.sh | sh 

    # For windows
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex" 

Then simply run the main function using uv to launch

    uv run main.py
